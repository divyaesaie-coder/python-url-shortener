# python-url-shortener
A simple Python URL shortener that generates short codes for long URLs and retrieves the original URL using the short code.
# Python URL Shortener

A beginner-friendly Python project that simulates a URL shortening service.

## Features
- Generate short URLs
- Store original URLs
- Retrieve original URL using short code
- Random short code generation
- Menu-driven console application

## Technologies Used
- Python 3
- random module
- string module

## How It Works
The program:
1. Takes a long URL from the user
2. Generates a random short code
3. Stores the original URL with the code
4. Retrieves the original URL when the short code is entered

## Concepts Used
- Dictionaries
- Random string generation
- Loops
- if-elif conditions
- User input handling
